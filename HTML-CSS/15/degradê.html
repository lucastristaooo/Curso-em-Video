Degradê em CSS

  • Usa-se "background-image em vez de background-color"
  • Usa-se o comando "linear-gradient" ou "radint-gradient" (para degradês redondos)
  • Escolhe a posição "to right" (opções: top, bottom, right, left)
  • Escolhe as cores
  • Define "height: 100%;" dentro das configurações globais da CSS (*{})
  • Usa-se o comando "background-attachment: fixed;" para o degradê não se restar ao longo da página
